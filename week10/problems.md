# Week 10: 데이터 타입 - 문제

## 📚 학습 범위
- 15장: 적절한 데이터 타입 선택 (CHAR vs VARCHAR, INT vs BIGINT 등)

---

## 문제

### Q1. CHAR vs VARCHAR
CHAR와 VARCHAR의 차이점을 설명하고, 각각 언제 사용해야 하는지 예시와 함께 설명하세요.

### Q2. VARCHAR 최대 길이
VARCHAR(255)와 VARCHAR(256)의 저장 방식 차이를 설명하세요.

### Q3. 숫자 타입 선택
INT, BIGINT, DECIMAL의 차이점과 각각의 사용 시나리오를 설명하세요.

### Q4. 날짜/시간 타입
DATETIME과 TIMESTAMP의 차이점을 설명하고, 각각 언제 사용해야 하는지 설명하세요.

### Q5. ENUM과 SET
ENUM과 SET 타입의 장단점을 설명하고, 사용 시 주의사항을 설명하세요.

### Q6. TEXT vs VARCHAR
TEXT와 VARCHAR의 차이점을 설명하고, 언제 TEXT를 사용해야 하는지 설명하세요.

### Q7. NULL과 저장 공간
NULL을 허용하는 컬럼과 허용하지 않는 컬럼의 저장 방식 차이를 설명하세요.

### Q8. 실무/면접 질문
"새로운 테이블을 설계할 때 데이터 타입을 선택하는 기준과 고려해야 할 점들을 설명해주세요."
