# 📚 Real MySQL 8.0 스터디

## 📌 스터디 개요

- **기간**: 11주 (주 1회 모임)
- **대상**: MySQL 기초를 다지고 싶은 백엔드 개발자
- **목표**: 실무 필수 개념 + 면접 대비

---

## 📖 커리큘럼

**Week 1: MySQL 아키텍처**

- 04장: MySQL 엔진과 InnoDB 스토리지 엔진 구조
- InnoDB 버퍼 풀, 언두 로그, 체인지 버퍼 등

**Week 2: 트랜잭션과 잠금 ⭐**

- 05장: ACID, 격리 수준, InnoDB 잠금 메커니즘, 데드락

**Week 3-4: 인덱스 ⭐ (2주)**

- 08장: B-Tree 인덱스, 클러스터링/세컨더리 인덱스
- 복합 인덱스, 인덱스 스캔 방식, 커버링 인덱스

**Week 5: 옵티마이저**

- 09장: 쿼리 실행 절차, 조인/서브쿼리 최적화

**Week 6: 실행 계획 ⭐**

- 10장: EXPLAIN 완벽 분석, type/Extra 컬럼 해석

**Week 7-8: SELECT 쿼리 작성 (2주)**

- 11.3장: MySQL 연산자와 내장 함수
- 11.4장: SELECT, JOIN, GROUP BY, ORDER BY 최적화
- 11.4.12장: 윈도우 함수 (ROW_NUMBER, RANK 등)

**Week 9: INSERT/UPDATE/DELETE & DDL**

- 11.5-11.6장: DML 최적화
- 11.7장: 테이블/칼럼/인덱스 변경, 온라인 DDL

**Week 10: 데이터 타입**

- 15장: 적절한 데이터 타입 선택 (CHAR vs VARCHAR, INT vs BIGINT 등)

**Week 11: 복제(Replication) & 종합 리뷰**

- 16장: Master-Slave 복제 개념
  - 복제 아키텍처와 동작 원리
  - 읽기/쓰기 분리 전략
  - Replication Lag 문제
- 전체 복습 + 실전 문제 풀이
